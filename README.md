## HousIe-game-board-

A animated web application for generating and tracking numbers for Houise. Built with a Python Flask backend and a responsive, wide-grid frontend.

##  Features

- Zero Repeats: Logic ensures numbers 1-90 are drawn only once per session.
- Animated UI: Smooth CSS transitions and JavaScript-driven number rolling effects.
- Wide Layout: 15x6 grid optimized for large screens and projectors.
- Latest Highlight: The most recent number is highlighted with a gold glow.

##  Tech Stack

- Backend: Python, Flask
- Frontend: HTML5, CSS3 (Grid & Flexbox), JavaScript
